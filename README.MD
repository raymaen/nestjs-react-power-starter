### Start

##### npm run install:all && npm start

### Nest CLI

##### creates new module

`nest g mo <name> --no-spec`

##### creates new controller

`nest g co <name> --no-spec`

##### creates new service

`nest g s <name> --no-spec`

##### creates new resolver

`nest g r <name> --no-spec`

##### creates new interface

`nest g interface <name>`
